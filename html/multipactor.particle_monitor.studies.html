multipactor.particle_monitor.studies module

Created on Mon Jul 10 15:55:49 2023.

@author: placais

In this module, we define functions to visualise the emission energy, the collision energy or the trajectories of the particles stored in a ParticleMonitor.

plot_collision_angles(particle_monitor: ParticleMonitor, bins: int = 100, hist_range: tuple[float, float] = (0.0, 90.0), **kwargs) → Figure

Compute and plot a particles impact angle histogram.

Parameters:

  • particles (ParticleMonitor) – particles

  • bins (int) – bins

  • hist_range (tuple[float, float]) – hist_range

  • check_collisions (bool, optional) – Check how many particles intersected the mesh, and how many particles intersected the mesh several times. The default is False.

Return type:

Figure

plot_collision_energies(particles: ParticleMonitor, bins: int = 1000, hist_range: tuple[float, float] = (0.0, 10000.0)) → Figure

Plot all the collision energies as histogram.

  • bins (int, optional) – Number of bins in the histogram. The default is 1000.

  • hist_range (tuple[float, float], optional) – Holds first and last energies of the histogram. The default is (0., 1e4).
